	Two functions, $h$ and $g$, are given
	\[
	h(x)=\frac{x^2-4}{4},\  1 <x < 3

	\[
	g(x)=x-2,\  1 < x < 3
	\]

	g(x) \le f(x) \le h(x),\ 1 < x < 3
